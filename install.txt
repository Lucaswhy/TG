Guia de instruções para meus amigos super inteligentes!!!

Primeiramente, vocês vão baixar os arquivos do github(todos)
E vão criar uma pasta, de preferência, na área de trabalho dos senhores.
Então, vocês vão deixar a pasta como está organizada na pastinha + arquivos
exemplo:
-Pasta1(module_nodes)
-Pasta2(views)
-Pasta3(public)
rotas.js
criandotabelas.js

PS: Deixar o nome das pastas exatamente igual.
PS2: Se por acaso, durante a instalação abaixo não funcionar por já existir ou algo assim,
tentem instalar depois de excluir a pasta module_nodes. (Ela é gerada quando se installa
os frameworks)
---------------------
CMD

As demais coisas feitas aqui, são no CMD, então, abram o cmd e façam o caminho até a pasta
que vocês deixaram os arquivos do git
exemplo:
cd documents
cd tg
------------
INICIANDO SERVIDOR

APÓS INSTALAR TUDO ABAIXO, PARA TESTAR O SERVIDOR USE NO DIRETÓRIO CORRETO DO CMD

nodemon rotas.js

deve retornar: Servidor Rodando.
Banco de dados funcionando.

Após isso, para iniciar, digite no chrome
localhost:8081
---------
"EDITORES DE TEXTO"

Não necessariamente, mas vocês vão precisar de um bom editor de texto tanto pro html
como pro back-end
Eu uso o SUBLIME TEXT(https://www.sublimetext.com/3) para o html
e o Visual Studio Code para o de mais. Também pode ser usado o ATOM (https://atom.io/)
Mas aí eu nunca usei.
--------------------------
Node.Js

Só baixar.
Link: https://nodejs.org/en/
Versão LTS da esquerda
----------------------

Node.JS Express
(Framework que faz as rotas do servidor!)

ATENÇÃO, DEIXAR O CMD NO DIRETÓRIO QUE ESTÁ AS PASTAS(como eu falei na parte do CMD)

apenas digitar e dar enter:
npm install express --save

Com isso, se vocês quiserem iniciar o servidor para testar é só usar

node rotas.js

e digitar no chrome localhost:8081

-----------------------

Nodemon
(Framework que faz com que o servidor fique ligado e reinicie sozinho assim que uma
alteração seja feita no código)

digitar no cmd no diretório correto:

npm install nodemon -g

--------------------------
mySQL

https://dev.mysql.com/downloads/mysql/

clicar em GO TO DOWNLOAD PAGE embaixo do windows installer
baixar o primeiro

No installador selecionar

Server only

Dar next até aparecer Config Type: Development Computer
(certifique-se que esteja nesse mesmo ou em Development Machine)
(muda o nome conforme a versão do installador)

Coloque para iniciar com o windows

senha: 123456

depois

Clicar com botão direito em meu computador>propriedades
Configurações avançadas do sistema>variáveis do ambiente
E clique em novo

Você vai pegar e copiar a pasta onde está os arquivos do TG
e clicar em novo e colar.

Depois, você vai no cmd(pode ser em qualquer diretório)

digite:

mysql -u root -p

coloque o password

e digite de novo:

CREATE DATABASE CAP

dê um SHOW DATABASES para confirmar se foi criada.
-----------------------
Sequelize
(framework do banco de dados)

Só digitar no CMD no diretório correto:

npm install --save sequelize

depois digitar:

npm install --save mysql2
------------------------
HANDLEBARS
(framework para template do html)

Só digitar no CMD no diretório correto:

npm install --save express-handlebars
-------------------------
Body-parser
(Framework para trabalhar com exibição dos dados dos formulários)

Só digitar no CMD no diretório correto:

npm install --save body-parser
